# ICC Discord Bot
An Open Sourced Discord bot written in Python(discord.py) for [ICC Open Source Competition](https://icc.devfolio.co/).<br>
**ICC Discord Bot** is a moderation bot with some Fun, Music, Utility, Memes & Game commands.
